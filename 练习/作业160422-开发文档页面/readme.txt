# 工作要求

按照设计图，完成页面制作。
说明：要符合设计图，需要精确测量图中元素的像素尺寸，熟悉PS等做图工具的小伙伴

# 操作规范

你可以看到 project-devdoc 项目文件夹，其中有基本的结构、以及切好的图片。
请在项目文件夹内进行修改，按要求完成工作。注意：请不要将设计图文件放入项目内。
完成后，将 project-devdoc 压缩成 zip 压缩包上传。
压缩包命名规范： [姓名]-开发文档页面.zip
